C++ was first started in 1983 as an extension of the C language. It is an object-oriented programming language designed to create lean, efficient code making it easier to work with on large scale projects.

C++ is considered to be an intermediate-level language due to its mixture of high- and low-level features, making it a great introductory programming language. C++ is used for a wide variety of projects such as computer programs, games, and applications just to name a few.
